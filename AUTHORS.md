## Contributors

This project follows the [All Contributors](https://allcontributors.org/) specification.

Contributions of any kind are welcome! See [CONTRIBUTING.md](./CONTRIBUTING.md).

## Becoming a Contributor

If you've contributed to this project, you can add yourself using the [All Contributors Bot](https://allcontributors.org/docs/en/bot/overview) by commenting on an issue or PR:

```
@all-contributors please add <username> for <contribution types>
```

## Reporting Contributions

The following contribution types are recognized:

| Type | Description |
|:-----|:------------|
| code | Bug fixes, features, refactoring |
| doc | Documentation improvements |
| test | Test additions or corrections |
| bug | Bug reports that lead to fixes |
| ideas | Ideas and feedback |
| project | Governance, CI/CD, tooling |
| data | AI metadata, assets |
| content | Content improvements |
| design | UI/UX feedback |
| translation | Localization efforts |
